Installation Guide

First Time Setup?
Follow these steps to install the required dependencies for face recognition.

1 Install Dependencies

Install required Python packages in your bench environment:

cd /path/to/your/bench
source env/bin/activate
pip install opencv-python pillow numpy

2 Install System Dependencies

For Ubuntu/Debian systems, install OpenCV system libraries:

sudo apt-get update
sudo apt-get install -y libopencv-dev libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev

3 Restart Services

Restart your bench to load the new packages:

bench restart

4 Verify Installation

Use the "Refresh Status" button above to check if everything is working correctly.
