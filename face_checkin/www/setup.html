Face Check-in Setup

Install dependencies for facial recognition functionality

Dependencies Required: The face recognition system requires additional Python packages to function properly.
1

Access Your Server

Connect to your Frappe/ERPNext server via SSH or terminal access where the bench is installed.

2

Activate Bench Environment

Navigate to your bench directory and activate the virtual environment:

cd /path/to/your/bench
source env/bin/activate
3

Install Face Recognition Dependencies

Install the required Python packages for face recognition:

pip install face-recognition pillow numpy
Note: This may take several minutes as it needs to compile dlib and other dependencies.
4

Install System Dependencies (if needed)

If you encounter errors, you may need to install system-level dependencies first:

# For Ubuntu/Debian:
sudo apt-get update
sudo apt-get install build-essential cmake
sudo apt-get install libopenblas-dev liblapack-dev
sudo apt-get install libx11-dev libgtk-3-dev

# For CentOS/RHEL:
sudo yum groupinstall "Development Tools"
sudo yum install cmake
sudo yum install openblas-devel lapack-devel
5

Restart Your Bench

After installation, restart your bench to load the new dependencies:

bench restart
